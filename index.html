Projetos

  • iFood Web Scraping & EDA

    Maio/2021

    O projeto envolveu coleta de dados do site do iFood sobre diversos restaurantes, por meio de tecnologias de Web Scraping em Python. Ademais, realizou-se a limpeza dos dados adquiridos. Por fim, realizou-se uma análise exploratória em R, documentada em R Markdown.

  • @RegressML

    Mar/2021

    Um bot no Twitter que publica previsões de ações com Machine Learning. Desde o Web Scraping para a coleta de dados diários da internet, treino de 7 modelos de regressão - Stochastic Gradient Descent, Ridge Regression, Support Vector Regressor, K-NN Regressor, Random Forest Regressor, Ada Boost e Multi-Layer Perceptron - e seleção dos melhores hiperparâmetros, até a publicação das 3 previsões mais confiáveis. Todos os dias - na sua timeline.

  • ENEM - Python e R

    Fev/2021

    Projeto de Análise Exploratória de Dados do ENEM. Notável que as escolas particulares brasileiras têm melhores performances que as públicas. Ademais, notou-se que as escolas técnicas são as melhores dentre as públicas. Discussões sobre como os ambientes determinam nossos resultados.
